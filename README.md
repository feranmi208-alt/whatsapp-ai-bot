# WhatsApp AI Bot

A fun, conversational WhatsApp bot that replies automatically — even when you're offline!

## Features
- Uses personal WhatsApp (not Business)
- Replies 24/7 on Render.com
- No coding experience needed
- Fun, witty responses

## Setup
1. Deploy to Render.com
2. Scan QR code with your phone
3. Turn off your phone — bot keeps working!
